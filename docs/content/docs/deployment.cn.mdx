## Docker

```sh
docker compose up --build
```

`docker-compose.yml` 里两种凭证方式：`MULERUN_TOKEN` 环境变量，或挂
`~/.config/mulerun` + `~/.mulerun` 到容器。

## 持久化 job 存储

默认 jobs 存内存，重启会丢。生产环境建议持久化：

```sh
# 本地 libsql 文件（纯 Go 驱动，零依赖）
CLI2API_JOBSTORE_DSN=file:/var/lib/cli2api/jobs.db ./cli2api

# 远端 Turso / sqld
CLI2API_JOBSTORE_DSN="libsql://your-db.turso.io?authToken=$TURSO_TOKEN" ./cli2api
```

DSN 的鉴权部分在启动日志里会自动脱敏，可以放心写入日志聚合系统。

## 反向代理（nginx）

流式端点要关掉 buffering，否则 SSE 和音频流会被卡住：

```nginx
location /v1/ {
    proxy_pass http://127.0.0.1:51222;
    proxy_buffering off;          # SSE + 音频流式
    proxy_read_timeout 600s;      # 长轮询 / 同步图像最长 5 min
    client_max_body_size 64M;     # multipart 图像编辑
}
```

## 文档站（本站）

本文档站位于 `docs/`，是**静态导出**的 Next.js 应用（`output: 'export'`）——
构建产物是纯静态的 `out/` 文件夹，无需服务端运行时。

**Cloudflare Pages**（推荐——免费、全球 CDN）：

```sh
cd docs
pnpm install && pnpm build      # → ./out
npx wrangler pages deploy out
```

或在 Cloudflare 控制台连接 GitHub 仓库，root directory 设 `docs`，build command
设 `pnpm install && pnpm build`，output directory 设 `out`。

静态托管下的语言路由由 `public/_redirects` 处理（`/` → `/en/`）；`/en/*` 和
`/cn/*` 已预渲染。搜索是客户端的（Orama 索引烤进 `out/api/search`，中文用
mandarin 分词），不依赖服务端。

**Vercel / Netlify / 任意静态托管** 也行——指向 `docs/out/` 即可。
